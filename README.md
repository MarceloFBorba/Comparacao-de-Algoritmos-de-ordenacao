# comparacao-de-Algoritmos-de-ordenacao
Um programa na linguagem C, na qual Implementa os Algoritmos de ordenação bolha, inserção, seleção, mergesort e quicksort., e Faz uma comparação de tempo entre eles utilizando a função clock() da biblioteca time.c. Foi gerado vetores de tamanhos 1000, 10000, 100000, 200000, 300000, 400000, 500000 com valores de inteiros crescentes, decrescentes e embaralhados. Foi usado o Algoritmo de Fisher-Yates para poder embaralhar, o vetor foi alocado com malloc() e desalocado com free(). O código deverá gerar três tabelas uma para cada situação de crescente, decrescente e embaralhado.
